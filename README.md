# Project Name

DPI Guatemala Generator

## Installation

1. Clone the repository
2. Create a virtual environment
```bash
python -m venv venv
```
3. Install the requirements
```bash
pip install -r requirements.txt
```


## Usage
    
```bash
python3 dpiDummy.py generar-dpi-dummy
```

## Compiling
```bash
python3 setup.py py2app -A
```
